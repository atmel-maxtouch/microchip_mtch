This repository holds the Linux driver, configuration, build and device tree file example file for the MTCH Touch Sensor chips.
The Linux driver mchp_mtch.c file supports the MTCH3380P and MTCH3240P family of touch sensors offering high speed acquistion channels.

These channels can be configuration as buttons/keys or groups to create sliders or wheels.

The MTCH3380P and MTCH3240P are part of the maXTouch family of touch controller chips using enhanced algorithms for noise avoidance, water tolerance and touch performance.
These algorithms are built into the firmware of the chip to provide robust buttons/wheels and sliders solutions.
